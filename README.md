# YouTube API Tryout

    Implementing some of the react features using Youtube API.

# SetUp

    Vite+React
    TailwindCSS
    React-Router-Dom
    Redux

# Layout

    App
        -Header
        -Body
            -Sidebar(show/Hide Toggle)
            -MainContainer
                -Button List
                -Video Container
                    -Video Cards (Click on video --> Watch Page)
            -MainContainer(search)
                -Button List
                -Search Container
                    -Search Cards (click on video --> Watch Page)

# Concepts

    Search Suggestions on input with Debouncing and caching results with redux
    N level recursive rendering for Comments
    API Polling to simulate Live chat locally
